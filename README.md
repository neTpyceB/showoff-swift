# Daily Notes

iOS SwiftUI app for create, edit, and delete notes.

## Requirements

- Xcode 26.4
- iOS Simulator 26.4

## Run

Open `DailyNotes.xcodeproj` in Xcode and run the `DailyNotes` scheme on an iPhone simulator.

## Test

```sh
xcodebuild test -project DailyNotes.xcodeproj -scheme DailyNotes -destination 'platform=iOS Simulator,name=iPhone 17,OS=26.4'
```
