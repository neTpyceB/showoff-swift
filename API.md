# API

## Daily Notes

Local operations:

- Create note.
- Edit note.
- Delete note.

## Weather Companion

External API:

- Geocoding: `https://geocoding-api.open-meteo.com/v1/search`
- Forecast: `https://api.open-meteo.com/v1/forecast`

Local operations:

- Add favorite place.
- Delete favorite place.
- Load current weather.
- Load hourly forecast.
