# Commands

## Build

```sh
xcodebuild build -project DailyNotes.xcodeproj -scheme DailyNotes -destination 'platform=iOS Simulator,name=iPhone 17,OS=26.4'
```

## Test

```sh
xcodebuild test -project DailyNotes.xcodeproj -scheme DailyNotes -destination 'platform=iOS Simulator,name=iPhone 17,OS=26.4'
```
